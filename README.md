# Docker command

### build service:

```sh
$ docker-compose build
```
### start service:

```sh
$ docker-compose up -d
```
### Stop service:

```sh
$ docker-compose down
```

### Access a running container

```sh
$ docker-compose exec $SERVICE_NAME bash
```

### start only 1 service

```sh
$ docker start $service id
```

### stop only 1 service

```sh
$ docker start $service id
```
